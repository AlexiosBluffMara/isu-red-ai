Developer API

Programmatic Access

RESTful endpoints. Search, ask, explore.

GET /api/search?q=query&k=10
Semantic vector search over all indexed papers.
{ "query": "rural education outcomes", "results": [...], "count": 10 }
GET /api/ask?q=question&k=8
RAG-powered Q&A with synthesized answers and sources.
{ "answer": "ISU research shows...", "sources": [...], "model": "gemini-2.5-flash" }
GET /api/stats
Pipeline stats — paper count, chunks, subjects, index metadata.
{ "total_chunks": 193653, "embedding_dim": 3072, "model": "gemini-embedding-2", "status": "operational" }
GET /
Web UI — the search interface, served via Jinja2 + FastAPI.
curl http://localhost:8000/ # Returns the ISU ReD AI search UI # FastAPI + Jinja2 application
